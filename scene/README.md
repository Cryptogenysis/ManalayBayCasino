# Manalay Bay Casino
Play slots in Decentraland. Place bets with MANA. Up to 20x payout. Based on the classic Liberty Bell slot machine algorithm.

## Start payout API
```
cd api/
npm install
nodemon server.js
```

## Start local Decentraland preview
```
cd scene/
npm install
dcl start
```
